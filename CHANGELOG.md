  - Bangumi API 集成
  - 自动 qBittorrent RSS 规则生成

- 🚀 **技术特性**
  - 多语言标题处理
  - 智能匹配算法

  - 缓存机制

- 🖥️ **GUI 编辑器**
  - Python tkinter 界面
  - 规则删除和编辑功能
  - 跨平台支持
  - 剪贴板复制

- 🔧 **开发工具**
  - GitHub Actions 自动化构建
  - 多平台编译支持
  - 完整的测试套件

### 支持的平台
- Windows (x86_64, i686)
- Linux (x86_64, aarch64)
- macOS (x86_64, aarch64)

### 配置要求
- Rust 1.70+
- Python 3.8+ (GUI 编辑器)
- DeepSeek API 密钥
